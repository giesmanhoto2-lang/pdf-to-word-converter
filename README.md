# 📄 Conversor PDF para Word

Um conversor simples, grátis e rápido para converter PDF para Word (DOCX), 100% funcional no seu navegador!

## ✨ Características

- ✅ **Completamente grátis** - Sem custos, sem planos
- ✅ **100% seguro** - Processamento local, seu PDF não sai do seu computador
- ✅ **Sem limites** - Converta quantos arquivos quiser
- ✅ **Rápido** - Conversão instantânea
- ✅ **Interface bonita** - Design moderno e intuitivo
- ✅ **Suporta drag & drop** - Arraste o arquivo para converter

## 🚀 Como Usar

1. Acesse o site (em breve disponível)
2. Arraste seu PDF para a caixa ou clique para selecionar
3. Aguarde a conversão
4. Baixe seu arquivo Word (DOCX)

## 💻 Tecnologias

- **PDF.js** - Leitura e extração de PDFs
- **docx.js** - Criação de arquivos Word
- **HTML5 + CSS3** - Interface responsiva
- **JavaScript vanilla** - Sem dependências desnecessárias

## 📋 Pré-requisitos

Nenhum! Tudo funciona direto no navegador.

## ⚠️ Limitações

- PDFs escaneados (imagens) podem não funcionar bem
- A qualidade de conversão depende da estrutura do PDF original
- Recomendado para PDFs com texto, não imagens

## 🔧 Configuração Local

Se quiser testar localmente:

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/pdf-to-word-converter.git

# Entre na pasta
cd pdf-to-word-converter

# Abra com um servidor local (Python)
python -m http.server 8000

# Ou com Node.js (se tiver http-server instalado)
http-server

# Acesse http://localhost:8000
```
